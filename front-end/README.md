## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app (Don't use this! Expo Go doesn't support our Google Login mechanism. Look at step 3)

   ```bash
    npx expo start
   ```

   Start the app in web

   ```bash
   npx expo start --web --https
   ```

3. Start the app

   ```bash
   npx expo run:ios
   ```

4. Run formatter and linter
   ```bash
   npm run lint
   ```
